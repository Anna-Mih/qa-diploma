# Перечень используемых инструментов с обоснованием выбора
_IntelliJ IDEA_ 1. - среда разработки для написания тестов (содержит в себе широкий функционал).
_Java 11_ 1. - язык программирования для написания автотестов.
_Gradle_ 1. - инструмент автоматизации сборки и управления зависимостями, подгружает автоматически прописанные зависимости.
_Selenide_ 1. - фреймворк для автоматизированного тестирования веб-приложений на основе Selenium WebDriver. Автоматически открывает и закрывает браузер, стабильный проход тестов. Совместим с Allur, автоматически делаем скриншоты падающих тестов.
_Allure_ 1. - фреймворк для создания простых и понятных отчётов автотестов. Создает визуальную картину прохода автотестов.
_Docker_ 1. - программное обеспечение для автоматизации развёртывания и управления приложениями в средах с поддержкой контейнеризации. Удобен для подключение баз данных и установки заглушек для приложения без установки специфического софта.
_GitHub_ 1. - крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Удобен для сохранения сделанной работы, с возможностью отката к предудыщей версии проекта.
